dynamodb_stream_router.router.StreamRouter¶

class dynamodb_stream_router.router.StreamRouter(*args, threaded: bool = False, **kwargs)¶
__init__(*args, **kwargs)¶

Provides routing of Dynamodb Stream records to Callables based on record content and/or truthy functions that may inspect the record

Keyword Arguments

  • threaded: (``bool`): If True then each record will be processed in a separate thread using ThreadPoolExecutor
