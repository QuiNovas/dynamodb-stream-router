dynamodb_stream_router.conditions.Old¶

class dynamodb_stream_router.conditions.Old(key: str)¶

Creates an Expression that evaluates against the old image of a record. Expressions can be grouped together using the Group class and be and’d and or’d together using ‘&’ and ‘|’.

Arguments

  • key: (str): The key in the old image to inspect
