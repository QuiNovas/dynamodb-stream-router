dynamodb_stream_router.conditions.Expression¶

class dynamodb_stream_router.conditions.Expression(exp)¶

A chain of Key, Old, New, HasChanged objects that are formed to create a final expression that can be evaluated against a stream record. It doesn’t really make sense to create an object of this type directly. It is a byproduct of creating an expression of statements using the above mentioned classes. It’s main purpose is to facilitate the evaluation of those statements.

An example expression:

from dynamodb_stream_router.conditions import Old, New

exp = Old("pk").eq("foo") & New("count").lt(10)
exp.record = {"old": {"pk": "foo"}, "new": {"count": 5}}

exp.evaluate()
# returns True
